# Offside Studios

Marketing site for **Offside Studios** — a web development studio building backend, frontend, branding and engineering for venture-backed founders, independent startups and SaaS companies.

Static site (HTML / CSS / vanilla JS) featuring an animated ASCII running stickman in the hero.

## Structure
- `index.html` — page markup
- `styles.css` — styling
- `script.js` — ASCII runner animation + marquee + contact form

## Deploy
Static — deploys on Vercel with zero config.
